bSlide
======

bSlide is an images slider plugin for jQuery.

Features
--------

Preserves the content of the DOM object.

Example
-------

Javascript:

    $('#target').bSlide();

HTML:

    <div id="target">
        <images>
            <li>image1.jpg</li>
            <li>image2.jpg</li>
        </images>
        <p>Some content...</p>
    </div>
